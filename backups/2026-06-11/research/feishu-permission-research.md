## 📝 飞书云文档权限体系

### 1. 文档分享权限类型
飞书云文档支持以下权限级别：

- **可管理**：拥有者可设置文档权限、管理协作者
- **可编辑**：可以编辑文档内容
- **可评论**：只能评论，不能编辑
- **可阅读**：只能查看，不能评论或编辑
- **无权限**：无法访问文档

### 2. 分享范围设置

飞书文档支持多种分享范围：

- **仅自己**：文档私密
- **组织内成员**：企业内可访问
- **互联网上任何人**：公开访问（可设置密码）
- **指定人**：仅限特定用户/群组

## 🛠️ 权限管理操作指南

### 场景1：给文档添加协作者

**适用情况**：需要将文档分享给团队成员编辑

**操作步骤**：
1. 打开飞书文档
2. 点击右上角「分享」按钮
3. 搜索添加成员/群组
4. 设置权限级别（查看/评论/编辑/管理）
5. 点击「完成」

**API方式**（已有权限支持）：
```
POST https://open.feishu.cn/open-apis/drive/v1/permissions/{token}/members
```

### 场景2：调整已有成员的权限

**适用情况**：需要提升或降低某成员的文档权限

**操作步骤**：
1. 打开文档，点击「分享」
2. 找到对应成员
3. 点击权限下拉菜单
4. 选择新的权限级别

**API方式**：
```
PUT https://open.feishu.cn/open-apis/drive/v1/permissions/{token}/members/{member_id}
```

### 场景3：移除文档成员

**适用情况**：团队成员离职或不再需要访问

**操作步骤**：
1. 打开文档「分享」设置
2. 找到对应成员
3. 点击「移除」按钮

**API方式**：
```
DELETE https://open.feishu.cn/open-apis/drive/v1/permissions/{token}/members/{member_id}
```

### 场景4：设置文档为组织内公开

**适用情况**：需要在团队内部分享但不对外公开

**操作步骤**：
1. 点击「分享」按钮
2. 开启「组织内成员可访问」
3. 设置默认权限级别（建议设置为「可阅读」）

### 场景5：设置文档外链分享

**适用情况**：需要分享给外部人员或客户

**操作步骤**：
1. 点击「分享」按钮
2. 开启「互联网上获得链接的人可访问」
3. 设置权限和密码（可选）
4. 复制链接分享

## 💡 建议方案

### 方案A：标准化权限模板
为不同类型的文档设置标准化权限：

| 文档类型 | 建议权限设置 |
|---------|------------|
| 公司制度文档 | 组织内可阅读 |
| 项目协作文档 | 项目组成员可编辑 |
| 客户资料文档 | 指定人可访问 |
| 个人工作文档 | 仅自己或直属上级可查看 |

### 方案B：定期权限审计
建议每月进行一次文档权限审计：
1. 列出所有共享文档
2. 检查离职人员是否还有权限
3. 清理不再需要的共享
4. 更新文档责任人
