# 公众号图文策划流程（小红书方向）

**触发条件：** 吕总需要小红书图文内容

**参与角色：**
- 文案虾（Content Master）：产出内容方案
- 虾米（CLO）：批判讨论、融合优化
- 吕总：最终确认

**执行步骤：**

### 步骤1：文案虾产出初版
- 确定选题角度
- 设计图文结构（P1封面 → P6解决方案）
- 撰写正文文案
- 标注画面描述

### 步骤2：虾米批判审核
检查维度：
- ✅ 封面真实感（不能太像广告）
- ✅ 情绪链条完整性
- ✅ 身份暗示清晰度
- ✅ 解决方案软硬度
- ✅ 数据真实性

### 步骤3：融合优化
- 整合文案虾创意 + 虾米批判
- 生成最终图文方案
- 准备即梦AI生图提示词

### 步骤4：生图执行
```bash
cd ~/.openclaw/workspace/skills/jimeng-ai
npx ts-node scripts/text2image.ts "提示词" --version v31 --ratio 3:4
```

### 步骤5：交付
- 6张图插入飞书文档
- 图文方案确认
- 发布计划制定

**风格规范：**
- 小红书：3:4竖图，现代职场漫画风
- 色调：冷色调工业蓝灰 + 红色/橙色点缀
- 人物：写实比例但表情夸张

**已确认选题：**
1. 《10分钟能修好的活，他拖了6天，我还得请他喝酒》（维修工揭秘视角）
2. 《开工厂15年，去年才算明白这笔账》（老板反思视角）
