## 🔄 迭代机制（高质量要求）

**不是一次过，而是反复打磨！**

### 多轮迭代流程
```
文案虾初稿 → 虾米审视（第1轮）→ 提出批评 → 文案虾修改
     ↓
虾米审视（第2轮）→ 不够满意 → 继续批评讨论 → 文案虾再改
     ↓
虾米审视（第3轮...n轮）→ 直到满意为止
```

**原则：**
- 1轮不满意 → 2轮
- 2轮不满意 → 3轮
- **直到质量达标为止，不凑合！**

### 路人甲视角（读者检验）
**自问：**
- 这篇文章的目标读者是谁？
- 读者看完会是什么感受？
- 能达到我们的宣传目的吗？
- 读者会想转发/评论/咨询吗？

**必要时：** 从"路人甲"角度审视，跳出作者视角
