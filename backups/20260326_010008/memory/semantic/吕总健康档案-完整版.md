## 核心算法与训练策略

### 目标 KPI
**最核心目标：减脂增肌**
- 配合司美格鲁肽使用
- 减脂的同时防止肌肉流失

### 具体目标
1. **左踝康复：** 通过等长收缩和针对性激活，固化踝关节稳定性
2. **背部增厚：** 利用 193cm 的臂展优势，打造产品经理的"厚实靠山"
3. **可持续运维：** 严禁过度训练导致系统崩溃（如之前的肠胃反复）

### 灰度发布逻辑
- **全量运行：** 体感回血 100%，执行高并发负重计划
- **降级发布：** 身体稍有不适（如脚踝微酸或肠胃轻微波动），自动切入"纯坐姿/低负压模式"
- **冷启动：** 长假或病后初愈，执行"Mobility（活动度）唤醒方案"
