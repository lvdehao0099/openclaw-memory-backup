# 每日早报生成流程

**触发时间：** 每天早8:00

**早报结构：**
1. 私人待办（最高优先级）
2. 高优先级工作待办
3. 今日必做工作
4. 空闲时间任务
5. 未来预警（提前3天提醒）

**数据来源：**
- 飞书虾米记忆备份库（主）
- 本地 memory/episodic/YYYY-MM-DD.md（备）

**生成步骤：**
1. 拉取飞书记忆库当日所有待办
2. 读取本地 memory 文件当日待办
3. 交叉对比，补全遗漏任务
4. 按优先级排序
5. 生成早报文本
6. 发送给吕总

**记忆同步规则：**
- 所有新增待办同时写入飞书+本地，双向同步
- 早报生成前强制拉取飞书记忆库最新数据
- 完成状态实时更新
