## 19:40 飞书表格创建铁律确认

**问题：** 表格单元格有空行

**根因：** PATCH body 格式错误
- ❌ 错误：`update_text_elements_request`
- ✅ 正确：`update_text_elements`

**正确方法：三步法**
1. 创建空表 → POST /children → 获取 table_id
2. 提取幽灵块 → GET /children → 获取每个单元格的空 text block ID
3. PATCH 覆盖 → `update_text_elements`（不是 `update_text_elements_request`）

**工具类：** `/root/.openclaw/workspace/scripts/feishu_table_manager.py`
