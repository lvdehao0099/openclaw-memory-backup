# 每日简报（早报）规则 - 2026-03-10更新版

## ⏰ 发送时间
**每天早上 8:00 整**（比原来提前半小时）

---

## 📋 简报固定结构

### 1. 🔴 私人待办（优先级最高）
- 放在第一条
- 个人生活事务
- 最高优先级处理

### 2. 🟡 工作待办（按优先级排序）
排序规则：
1. **#销售** - 客户拜访、跟进、签约
2. **#申报** - 协会申报、资质申请
3. **#宣传** - 公众号文章、内容运营、品牌传播
4. **#产品** - AI老师傅产品相关
5. **#项目** - 外包项目、实施交付

### 3. 🟢 注意提醒（提前1天预警）
包含内容：
- 未来24小时内的会议
- 客户拜访安排
- 截止日期提醒
- 其他时间敏感事项

**新增机制：** 提前1天提醒确认时间是否有变动

---

## 🔄 任务跟踪规则

### 任务未完成处理
1. **用户告知未完成** → 自动顺延到下一天早报
2. **用户告知截止日期** → 按截止日期设置提醒

### 截止日期提醒策略
| 时间距离 | 提醒频率 |
|----------|----------|
| 距离较长 | 提前3天第一次提醒 |
| 最后3天 | **每天提醒** |
| 最后1天 | 重点标注，多次确认 |

**直到用户明确告知"已完成"为止**

## 📝 简报格式示例

```
【虾米·2026-03-09 早报】

🔴 私人待办：
  - 联系修沙发师傅，约上门检修时间

🟡 工作待办：
  - 【#销售】苏浙市场拓展方案初稿截止
  - 【#申报】软件协会申报材料审核

🟢 提醒（未来24小时）：
  - 明天下午2点和苏州XX客户会议，已提前1天提醒，请确认时间无变动
```

---

## ⚠️ 重要提醒

1. **每天早上8:00准时发送**，绝不延误
2. **任务状态实时跟踪**，未完成的自动顺延
3. **截止日期严格监控**，最后3天每日提醒
4. **会议/拜访提前1天确认**，避免时间冲突
5. **用户确认"已完成"后才停止提醒**
